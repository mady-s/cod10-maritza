Nro Nombre del elemento Descripcion Ejemplo

fa-rotate-270 textarea Es un campo de entrada que puede tener multilinea y se define sus dimenciones mediante las propiedades cols y rows

  • Es un elemento que contiene una lista de opciones a legir

lista de tipos de input

Numero Tipo Descripcion Ejemplo
1 text Ingresar texto plano, permite el ingreso de todo tipo de caracteres
2 email Sirve para ingresar correos electrónicos pero a pesar de ser un texto plano, mediante la opcion de validar el formulario seasegura que lo ingresado tenga una @ y un .(punto)
3 number Sirve para ingresar numeros enteros y decimales
4 date Sirve para ingresar unicamente fechas, con un calendario de muestra
5 password Sirve para ingresar las contraseñas, oculta lo ingresado por medio de simbolos
6 time Sirve para ingresar las horas y minutos, mas no la fecha
7 color Sirve para ingresar un Input para seleccionar un color en base a una paleta de colores
8 checkbox Sirve para seleccionar un grupo de checkboxes, se puede seleccionar una o varios, van de la mano de su label
9 radio Sirve para seleccionar de un conjunto de obciones, solo se puede seleccionar uno a la vez mediante su propiedad name
10 file Sirve para subir un archivo local al servidor
11 reset Sirve para limpiar todo el formulario, limpia todos los campos, tiene que ir dentro de una etiqueta form para que limpie los campos
12 submit Sirve para enviar todos los campos del formulario, por defecto hará que se recargue la página(Ejemplo aplicarlo en la fila 11)
